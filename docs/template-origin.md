# Template Origin

This project was bootstrapped during Phase 2F from:

- Package: `com.deucarian.template.game.idle-auto-defense`
- Registry source: `https://raw.githubusercontent.com/Deucarian/Package-Registry/main/packages.json`
- Template URL: `https://github.com/Deucarian/Template-Game-Idle-Auto-Defense.git#main`
- Imported sample: `Basic Idle Auto Defense Game`

The starter scene and starter content were copied into `Assets/Moss/IdleAutoDefense` so the Moss game owns its product layer. The Deucarian package code remains installed through Package Manager and should not be edited from this game project.
