# Asset Flip Checklist

Start from the neutral imported template sample, then replace content intentionally:

- Replace enemy, weapon, projectile, and objective visuals under `Prefabs` or assigned authored assets.
- Tune stage, wave, upgrade, reward, offline, and monetization placeholder data under `Content`.
- Rename IDs into a real product namespace only when a real product theme exists.
- Keep reusable framework code in Deucarian packages.
